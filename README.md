## Descripción

El portafolio presenta información sobre Andrea Pineda, incluyendo una sección de bienvenida, una descripción personal, habilidades técnicas, proyectos destacados y formas de contacto. El diseño es responsivo y moderno, utilizando Bootstrap y animaciones con canvas.

## Características

- Diseño responsivo y moderno.
- Sección de bienvenida animada.
- Listado de habilidades técnicas.
- Muestra de proyectos realizados.
- Sección de contacto con enlaces a redes sociales y correo.
- Accesibilidad mejorada (enlaces y botones con texto discernible).
- Favicon personalizado.

## Tecnologías Utilizadas

- **HTML5**
- **CSS3** (incluyendo Bootstrap 5)
- **JavaScript**
- **Font Awesome** (íconos)
- **Google Fonts** (Inter)
- **SVG** (logo personalizado)

## Estructura del Proyecto

```
portfolio/
│
├── assets/
│   └── logo.svg
├── index.html
├── style.css
├── script.js
└── README.md
```

- `index.html`: Página principal del portafolio.
- `style.css`: Estilos personalizados.
- `script.js`: Animaciones y scripts personalizados.
- `assets/logo.svg`: Logo en formato SVG utilizado también como favicon.

## Instalación y Uso

1. **Clona el repositorio**  
   ```sh
   git clone <URL-del-repositorio>
   cd portfolio
   ```

2. **Abre el archivo `index.html` en tu navegador**  
   Puedes abrirlo directamente o usar una extensión de servidor local como Live Server en VS Code.

3. **Personalización**  
   - Modifica los textos, proyectos y datos de contacto en `index.html` según tus necesidades.
   - Cambia el logo en `assets/logo.svg` si lo deseas.

## Accesibilidad

- Todos los enlaces y botones incluyen texto discernible, visible o accesible solo para lectores de pantalla.
- El sitio es navegable con teclado y compatible con lectores de pantalla.
